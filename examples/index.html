Leap Motion examples

Here are a few examples using the javascript api.

DOM based visualizer
This is a DOM based visualizer of the data contributed by @juliangarnier (big thank you!)
3D Globe
3D Spinnable globe, contributed by @rtilton1. Ridiculously pretty.
Three.js visualizer
Very basic visualizer of data using Three.js
Dumper
Displays JSON data coming from websocket
Camera manipulation demo
Allow movement of cube through grasping actions
Hue Saturation Color Space
Move through hue-saturation colorspace with palm position
Canvas Fingers
Show finger positions in the plane of the screen on an HTML5 canvas element
Canvas Spiral
Creates a spiral with canvas coordinate transformations using palm normal
Gesture JSON Print Out
Prints out gesture data in JSON
